Website portfolio pribadi ini dibuat untuk menampilkan informasi diri, pengalaman organisasi, serta project yang telah saya kerjakan selama belajar di bidang Rekayasa Perangkat Lunak (RPL). Website ini berfungsi sebagai media untuk memperkenalkan kemampuan dan hasil karya saya secara profesional.

Role
Web Developer

Overview

Website portfolio pribadi ini saya kembangkan menggunakan HTML, CSS, dan JavaScript sebagai media untuk menampilkan profil, pengalaman organisasi, serta project yang telah saya kerjakan. Website ini dirancang dengan tampilan yang sederhana, responsif, dan mudah diakses di berbagai perangkat. Melalui website ini, saya dapat menyusun dan menampilkan informasi secara lebih terstruktur, sehingga memudahkan orang lain dalam melihat kemampuan, pengalaman, dan perkembangan saya di bidang pengembangan web.
