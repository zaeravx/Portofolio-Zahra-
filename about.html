Siapa Saya?

Halo! Saya Zahra Fahira Rahman, seorang siswa SMKN 4 Bandung yang memiliki ketertarikan di bidang UI/UX Designer dan pengembangan web

Perjalanan saya di dunia digital dimulai saat saya mulai tertarik mempelajari bagaimana sebuah website dibuat. Saya masih dalam proses belajar dan terus mengembangkan kemampuan di bidang ini. Saat ini, saya mempelajari Figma, HTML, CSS, dan JavaScript untuk front-end, serta mulai memahami dasar back-end menggunakan PHP, Laravel dan MySQL. Saya juga mencoba membuat project sederhana untuk meningkatkan skill saya secara bertahap.
